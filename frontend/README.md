# EcoTrack – SDG 13: Real-Time Carbon Footprint Tracker

![EcoTrack](https://via.placeholder.com/800x400?text=EcoTrack+SDG+13)

## Live Demo

http://localhost:5173 (Run locally)

## Features

- Real-time CO₂ calculation
- Live leaderboard (Socket.IO)
- Activity logging (Transport, Food, Energy)
- Secure auth (JWT + httpOnly)
- Mobile-first + Dark Mode
- Reset footprint
- MongoDB + Express + React + Node

## Tech Stack

- **Frontend**: React, Vite, Tailwind v4, Socket.IO Client
- **Backend**: Node.js, Express, MongoDB
- **Real-Time**: Socket.IO
- **Auth**: JWT, httpOnly cookies

## Setup

````bash
# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npm install
npm run dev
````
